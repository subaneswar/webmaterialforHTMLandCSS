TABLES

  1. it is a combination of rows and columns
  2. main parent tag is table
  3. there are many child tags used in parent tag table
    1. tr( table row )
    2. td( table data )
    3. th( table heading )

    1. tr(table row):- if we wanted to create a rows we will be using this tag inside table tag
    2. td(table data):- suppose if we wanted to create a cell in rows we will be using this tag inside tr tag
    3. th(table heading):- this tag also can be used in tr tag
      1. the purpose of using this tag is to print the content in bold and it should aligned to center of the cell
      2. used for main heading in table

    • Attribute in table:- border,cellspacing,cellpadding
    • attributes used in th and td:- rowspan,colspan

Forms

  1. it is used to take the userdata from users
  2. form is the main parent tag
  3. all the children tags or forms elememnts are inline elements

Elements used in form

  1. label - it is a tag where we will be using it for naming a textbox so user will get to know what to fill in the textbox
  2. input - It is a tag used for creating a textbox on the UI, user can use this textbox to fill data
  3. textarea - it is a tag used to create a textbox but with a bigspace, generally textarea is for address data
  4. select - It is used to create drop down data in the form for users. It is a parent tag, Child tag used inside this tag is option tag
    1. option tag - It is used to give many option to select for the user in the form
