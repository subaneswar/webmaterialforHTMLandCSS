BACKGROUND PROPERTIES

Background color:-

this property is used to apply background color for the element whichever you choose

eg-> h1{ background color:"blue" }

background image:-

By default if you add backgrond-image will be repeating we can control by using this property the value for this property is "no repeat","repeat-x"(x-axis),"repeat-y"(y-axis)

eg-> section{ background-image : url("path of the image")
background-repeat : no-repeat or repeat-x or repeat-y }




background-repeat:-

By default if you add backgrond-image will be repeating we can control by using this property the value for this property is "no repeat","repeat-x"(x-axis),"repeat-y"(y-axis)

eg-> section{ background-image : url("path of the image")
background-repeat : no-repeat or repeat-x or repeat-y }


background-position

if we want to manage position of the background image default value of the property is left values are - right/left/top/bottom/center

eg-> section{ background-image : url("path of the image") backgrong-position : center/top/bottom/left/right }

eg-> section{ background-image : url("path of the image") backgrong-position : center/top/bottom/left/right }

background-attachment:-

if we want to fix the background this property is used values are scroll or fixed scroll is the default value

eg-> section{ background-image : url("path of image"); background-attachment : scroll/fixed; }

Gradient:-

There are combination of multiple colors we have 2 types of gradients:-

  1. Linear gradient
  2. Radial gradient
  1. Linear gradient

    2. the color will start from the top of the UI we can give no. of colors by default the color will start from top to bottom

    eg-> section{ height:100vh; width:100%; background : linear-gradient(red,gree,blue) }

  2. Radial gradient

    3. the color will start from the center of the UI we can give no. of values

    eg-> section{ height : 100vh; width : 100%; background : radila gradient(red,ble,green) }

  2. different style for dorder we have solid,dashed, dotted,double,groove

  4. border-color:- color for the border
  5. border-radius:- to apply some curves on edges of the border you can apply values pixels(px) or percentage(%)
  6. you can target one particular side of the border & apply different styles we have

    3. border-left
