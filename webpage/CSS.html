CSS

  • It is used to design and also to give proper alignment to the webpage.
  • there are 3 ways to use CSS:-
  1. Inline css
  2. Internal css
  3. External css
  1. Inline css:-
    • it is type of CSS where we will be applying style properties in the line itself
    • Inline css is applied using "style" attribute
    • for each tag we have to declare "style" attribute and the css properties
    • we can add n numbers of css properties using inline
  2. Internal css:-
    • It is a type of css where tag is used to apply CSS properties
    • style tag should written inside head tag
    • Inside style tag you need to target particular element and apply css properties
  3. External css:-
    • we have to create a sperate file while using external css
    • file extension for extrnal css file in ".css"
    • target an element and apply css properties
    • and also we have to link css file with respected html file
